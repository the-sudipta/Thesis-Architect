The everyday research mess

Research work is hard enough before the folders start fighting back.

Without a repeatable structure, every project slowly invents its own chaos. Notes drift away from PDFs, drafts lose their supporting figures, and useful meeting feedback hides inside random downloads.

Scattered files

Papers, notes, datasets, charts, and drafts land in separate places, so every review session starts with searching.

Repeated setup

Students rebuild the same folder layout for every thesis, article, seminar, or long-term study.

Weak handoff

Supervisors, collaborators, and future-you need predictable locations for literature, results, and revisions.

Lost context

Research questions, logs, gaps, and citation strategy are easy to forget when there is no home for them.

The answer

One topic name becomes a complete workspace.

The app creates numbered research sections, starter Markdown notes, BibTeX files, an Obsidian canvas, reusable templates, and a plain-text structure map. It skips anything already present, so rerunning it is gentle on existing work.

How to use it

The release binaries are simple on purpose.

Download

Get the right standalone binary for your operating system from GitHub Releases.

Launch

Run `thesis-architect` or double-click the Windows `.exe`.

Name

Type a short research topic or title, then press Enter.

Open

Find the new research folder in the current output folder.

Work

Add papers, data, notes, figures, references, and submissions in their proper places.

Folder map

Fifteen zones for the whole research lifecycle.

Click a zone to jump the explorer detail panel to that section.
